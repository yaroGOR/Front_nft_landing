# Description

One page React app with gap animations, swiper and react-confetti and typewriter effects.

# Technologies

React, JavaScript, HTML, CSS, styled-components, react-router, swiper, gsap

# Install

clone this repo with:

```bash
git repo clone yaroGOR/Front_nft_landing
```

open project  folder with “cd” in terminal and run 

```bash
npm install
```

```bash
npm start
```

note that you need node.js installed on your pc to run that commands
